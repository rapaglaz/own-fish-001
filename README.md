# Fish Config

My Fish shell setup. Has some shortcuts, theme, works with mise/atuin/zoxide/starship.

## Setup

```bash
git clone git@github.com:rapaglaz/own-fish-001.git ~/.config/fish
```

```bash
curl -sL https://raw.githubusercontent.com/jorgebucaran/fisher/main/functions/fisher.fish | source && fisher install jorgebucaran/fisher
```

```bash
fisher update
```

## Tools it uses

Works better if you have: [mise](https://mise.jdx.dev/), [atuin](https://github.com/atuinsh/atuin), [zoxide](https://github.com/ajeetdsouza/zoxide), [starship](https://starship.rs/), [bat](https://github.com/sharkdp/bat), [eza](https://github.com/eza-community/eza)
